早期的适配方案

定高，宽度使用百分比

多媒体查询方案（Media Query）

媒体查询可用于检测很多事情，例如：

  1. viewport(视窗) 的宽度与高度
  2. 设备的宽度与高度
  3. 朝向 (智能手机横屏，竖屏) 。
  4. 分辨率

使用多媒体查询可以在指定的设备上使用对应的样式替代原有的样式。

rem时代

相对长度单位指定了一个长度相对于另一个长度的属性。对于不同的设备相对长度更适用,其中有em，ex，ch，rem，vw，vh，vmin，vmax，%。这里主要说rem.

rem 是根 em（root em）的缩写，rem作用于非根元素时，相对于根元素字体大小；rem作用于根元素字体大小时，相对于其出初始字体大小。

rem的优点就是设定号rem的值后在不同设备显示效果比例是一致的。但是不能每个px自己计算转换成rem。
