发表于 2020-04-23 更新于 2021-07-15 分类于 DesignPattern

构造器模式

在经典的面向对象的编程语言中，构造函数是一种特殊的方法，用于在为其分配了内存后初始化新创建的对象。在 JavaScript 中，几乎所有事物都是对象，因此我们最常对对象构造函数感兴趣。

对象构造函数用于创建特定类型的对象-准备对象以供使用并接受参数，以便在首次创建对象时，构造函数可以使用该参数来设置成员属性和方法的值。
